# crawling-kurly

현재 개인 프로젝트로 Python 기반 데이터 수집 및 분석 시스템을 개발중

사용자가 원하는 식품을 입력하면 Selenium을 통해 Kurly에서 상품 영양 정보를 자동으로 수집하고, 
해당 데이터를 OCR을 통해 텍스트화한 뒤 SQLite / Oracle DB에 저장하는 구조를 구현하려 함

이후 수집된 데이터를 API 형태로 제공하고 React 기반 프론트엔드를 통해 사용자에게 알러지 정보나 
슈링크플레이션(용량 감소 등) 감지 결과를 알려주는 서비스로 확장할 계획
